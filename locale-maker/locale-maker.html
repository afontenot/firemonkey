Locale Maker 1.3
 

🇭 Help

🔘 Select Language
Select & load the locale from extension _locale if available
📥 Import
After selecting the language, import a locale from file
📤 Export
Export the current locale to {language-code}/messages.json (blank entries will be filled with default language)
⬇️ Paste
Usefull when copy/pasting from a translation API, on paste, if there are more than 3 lines of text, each line will fill the inputs starting from active input or the first one
👥 Share
Please share your translations to extension-translations
